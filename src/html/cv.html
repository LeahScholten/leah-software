Wie ik ben:

Ik ben een autistische Software Engineer. Als kind wilde ik graag games ontwikkelen,een website maken en robots bouwen. Toen ik 15 was kwam ik een ICT docent tegen die mij vertelde hoe ik het zelf kon leren. Eerst HTML en Python, daarna Java, Javascript, C#, C en C++.
Tijdens mijn studie heb ik ook nog de Function Block Diagram (FBD) en Structured Control Language (SCL) PLC programmeertalen geleerd om te gebruiken in TIA Portal, om PLC's aan te sturen.
Ik heb zelf tijdens mijn studie ook nog Rust geleerd. Sinds 2015 heb ik al mijn doelen bereikt en heb ik besloten om mij vooral te richten op systems engineering (OS, firmware, Embedded, CLI).
Door mijn autisme ben ik stil en graag alleen, maar ben ik ook erg gedreven als ik doe wat ik leuk vind. In een groepsgroote van maximaal 5 en met een duidelijk overzicht waar iedereen aan werkt, kan ik ook prima samenwerken. Helaas ben ik door mijn autisme wel erg gevoelig voor harde geluiden en mensen die door elkaar heen praten.
